# Gestión de Tareas

Aplicación Angular para gestionar tareas.

## Requisitos

- Node.js 20.x o superior (recomendado 24.x)
- npm 10.x o superior

## Instalación

```bash
npm install
```

## Ejecución

```bash
ng serve
```

## Tecnologías

- Angular 21.1.4
- TypeScript 5.9.3
- RxJS 7.8.2
- Bootstrap 5.3.3
